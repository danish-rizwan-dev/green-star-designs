# Green Star Designs & Consultants Pvt. Ltd.

## Premium Architecture & Engineering Website

A world-class, production-ready website for **Green Star Designs & Consultants Pvt. Ltd.** — a premium architecture, civil engineering, structural design, and construction consultancy firm based in Bettiah, Bihar, India.

---

## 🏗️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| **Next.js 15** | React framework with App Router |
| **TypeScript** | Type-safe development |
| **Tailwind CSS** | Utility-first styling |
| **Framer Motion** | Premium animations & transitions |
| **Lucide React** | Modern icon library |

## 🚀 Pages

### 1. Home Page (`/`)
- **Hero Section**: Full-screen with animated floating stats, gradient overlay
- **Stats Banner**: Animated counters (500+ Projects, 100+ Clients, etc.)
- **About Section**: Two-column with image and company intro
- **Expertise Section**: 8 service cards with hover animations (glassmorphism)
- **Timeline Section**: 5-step project process with animated timeline
- **Why Choose Us**: 8 feature cards on blue background
- **Featured Projects**: Filterable portfolio gallery with 6 projects
- **Testimonials**: 4 client testimonials with star ratings
- **CTA Section**: Full-width call-to-action with background image

### 2. About Us (`/about`)
- Company Overview with image
- Mission & Vision cards
- Core Values (6 values)
- Experience Timeline (2016-2026)
- Leadership Team section
- Statistics banner
- Professional Certifications

### 3. Services (`/services`)
- 8 detailed service cards with expandable content:
  - Overview, Benefits, Process, Deliverables
- FAQ accordion section (5 questions)
- CTA banner

### 4. Projects (`/projects`)
- Category filter (All, Residential, Commercial, Industrial, etc.)
- 9 project cards with hover effects
- **Modal Preview**: Click any project for detailed view with features

### 5. Contact (`/contact`)
- 4 contact info cards (Address, Phone, Email, Hours)
- Contact form with validation
- WhatsApp CTA button
- Emergency Consultation CTA
- Service area map (All India)

## ✨ Features

### Animations (Framer Motion)
- **Fade Up**: Elements animate in on scroll
- **Slide In**: Directional entrance animations
- **Counter Animation**: Number counting with easing
- **Hover Effects**: Cards lift and scale on hover
- **Stagger**: Sequential card animations
- **Modal Transitions**: Smooth open/close with scale
- **Page Transitions**: Smooth navigation

### Design Elements
- **Glassmorphism**: Frosted glass effects on cards
- **Gradient Overlays**: Hero and CTA sections
- **Gold Accents**: Premium feel throughout
- **Responsive**: Mobile-first, works on all devices
- **SEO Optimized**: Meta tags, Open Graph, structured data
- **Custom Scrollbar**: Branded scroll experience

### Interactive Components
- **Mobile Menu**: Full-screen overlay with staggered links
- **Service Expand**: Click to reveal detailed information
- **FAQ Accordion**: Smooth expand/collapse
- **Project Filter**: Category filtering with animation
- **Project Modal**: Detailed project preview overlay
- **Form Validation**: Client-side form handling
